# Global Flood Mapping

This project builds on the work of [MikeFHS](https://github.com/MikeFHS). The scripts in this repository are designed to create flood maps using multiple Digital Elevation Model (DEM) sources and the GEOGLOWS River Forecast System. The goal is to develop a comprehensive global flood mapping system that can provide accurate and timely flood forecasts and retrospective analysis.

Documentation is [here](https://global-flood-scripts.readthedocs.io/en/latest/).
